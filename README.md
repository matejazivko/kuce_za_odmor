Kratki opis funkcionalnosti: Aplikacija će služiti iznajmljivačima kuća za odmor za promociju istih. Sadržavati će slike kuća, osnovne informacije o kućama i kontakt iznajmljivača. Registrirani korisnici moći će pregledavati postojeće i dodavati nove slike i informacije o kućama, komentirati i putem kontakt obrasca kontaktirati iznajmljivače. Neregistrirani korisnici imati će mogućnosti samo pregledavati sadržaj aplikacije (slike, informacije o kućama i komentare).
